# GenTA — IR Deck
### AI Video Intelligence Platform

---

## 1. One-liner

**GenTA는 영상의 감정을 수치화하고, 그 수치로 최적의 영상을 자동 생성하는 AI 플랫폼이다.**

GPS가 위치를 좌표로 표현하듯, GenTA의 GACS 엔진은 영상의 감성을 좌표로 표현한다.
Runway와 Pika는 "영상을 만든다." GenTA는 "성과를 내는 영상을 만든다."

---

## 2. Problem

### 영상 콘텐츠 제작의 3가지 고통

**1. 비용**: 에이전시에 건당 100-500만원, 2-4주 소요. 교육기관, 프랜차이즈, 보험사는 매월 수십 개의 영상이 필요하지만 감당 불가.

**2. 일관성**: 브랜드가 원하는 톤, 색감, 감정을 매번 같은 수준으로 유지하는 것이 불가능. 담당자가 바뀌면 스타일이 바뀜. 프랜차이즈 100개 지점의 영상이 다 다름.

**3. 최적화 불가**: 어떤 영상이 "잘 되는" 영상인지 정량적으로 모름. A/B 테스트를 해도 "왜" 잘 됐는지 모름. 감이 아닌 데이터로 영상을 설계할 수 없음.

---

## 3. Solution: GACS + AI Video Pipeline

### GACS (Generative AI Coordinate System) — 해자

GACS는 영상의 감성을 7축 좌표로 수치화하는 자체 엔진.

| 축 | 의미 | 설명 비중 |
|---|---|---|
| PC1 | 공포 ↔ 기쁨 (Valence) | 27.8% |
| PC2 | 슬픔 ↔ 흥분 (Arousal) | 14.0% |
| PC3 | 공포 ↔ 슬픔 | 11.1% |
| PC4 | 감정 ↔ 집중 | 7.2% |
| PC5 | 신비 ↔ 분노 | 4.8% |
| PC6 | 따뜻함 ↔ 긴장 | 4.0% |
| PC7 | 평온 ↔ 진지함 | 2.5% |

- **7축 합계: 전체 감정 차이의 71.4% 설명**
- 모든 축 cosine similarity > 0.94 (통계적으로 안정)
- Russell Circumplex 모델이 데이터에서 자연 출현 (학술적 근거)
- PC3(공포↔슬픔 분화)은 기존 2D 감성 모델의 한계를 넘어서는 독자적 발견
- **라벨링 완료, 성과 데이터 연결 중**

### 왜 GACS가 해자인가

```
Runway/Pika/Sora:  프롬프트 → 영상 (생성)
GenTA/GACS:       목표 감성 → 최적 씬 조합 → 영상 → 성과 측정 → 최적화 (설계)
```

생성 모델(Veo, Sora, Runway)은 교체 가능한 부품. **어떤 감성의 씬을 만들 것인가**를 GACS가 결정한다. 생성 모델이 발전할수록 GACS의 가치는 올라간다.

## 4. Product

### GenTA Studio — AI 영상 제작 파이프라인

시놉시스 입력 → Claude가 대화형으로 기획 → 씬 분할 → 키 이미지 생성(나노바나나2) → 영상 생성(Veo 3.0) → 후보정(자막, 트랜지션) → 내보내기

**5단계 파이프라인:**
1. 💡 아이디어 (Claude 대화형 기획 + 하네스 구축)
2. 📋 시놉시스 (자동 채우기)
3. ✍️ 시나리오 (자연어 → AI 정리 → 대본)
4. 🎬 프로덕션 (이미지/영상 생성, 레퍼런스 라이브러리)
5. 🎛️ 후보정 (다중 자막, 트랜지션, 속도 조절, FFmpeg 렌더링)

### 하네스 엔지니어링 — B2B 핵심

"하네스"는 프로젝트 전체에 적용되는 브랜드 스타일 지침서.
- 비주얼 스타일, 색감, 카메라, 톤, 프롬프트 프리픽스
- 한 번 세팅하면 모든 영상에 동일 스타일 자동 적용
- 클라이언트의 브랜드 가이드라인을 AI가 읽고 하네스 자동 구축

**결과**: 시놉시스만 넣으면 브랜드에 맞는 영상이 자동으로 완성.

### AI 사서 어시스턴트 — B2G 확장

- 3D AI 사서 아바타 + 립싱크 TTS
- 대출 이력 기반 초개인화 도서 추천
- 도서 소개 영상 자동 생성
- GACS 데이터 파이프라인 연동 (모든 인터랙션 수치화)

## 7. Business Model

### 3가지 수익원

**1. 구축비 (일회성)**
- 클라이언트 브랜드 분석 → 하네스 세팅 → GACS 프로필 구성
- B2G: 기관별 협의 / B2B: 50만원 또는 셀프서브

**2. 월 구독 (반복 매출)**
- B2G: 관당 30-50만원/월
- B2B: 50만원/월 (영상 20건 포함)

**3. 생성비 (종량제)**
- 건당 $3-5 (초과 생성 시)
- 클라이언트가 많이 쓸수록 매출 증가
